Example 2

These buttons add some interaction. Notice how the custom properties are used. custom properties defined in the .example-2-button selector are available (through scope) in the .example-2-button:hover selector.

We can use an inline style to customize the values in the same way as example 1.

Important! In this example the value of --button-color changes in the :hover rule and the appearance of button changes. Changes to CSS custom properties are applied immediately!

That deserves repeating. Anytime the value of a custom property changes those changes are applied immediately.
